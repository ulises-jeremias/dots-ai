# Notion Integration

> Notion pages and databases through the MCP template.

---

## Best path

Use the Notion MCP template and a local API token.

## Setup

```bash
export NOTION_API_TOKEN="..."
```

Then register the Notion MCP server in your AI tool config using the template in `MCP`.

## What it enables

- Page access
- Database search
- Structured content workflows

```bash
dots-doctor
```
