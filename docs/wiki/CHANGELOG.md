> Version history for the dots-ai platform.

## Overview

This page summarizes significant changes to the workstation baseline. For the full changelog, see [`CHANGELOG.md`](https://github.com/ulises-jeremias/dots-ai/blob/main/CHANGELOG.md) in the repository root.

---

## Format

Changes follow [Keep a Changelog](https://keepachangelog.com/) format with these categories:

- **Added** — new features
- **Changed** — modifications to existing features
- **Deprecated** — features that will be removed
- **Removed** — features that were removed
- **Fixed** — bug fixes
- **Security** — vulnerability fixes

---

## Recent highlights

The workstation platform is under active development. Key areas of recent work:

- Expanded skill catalog with workflow and dev companion skills
- Atomic Best Practices skills for PRD/TRD/ADR, work items, planning, meetings, decisions, agreements, spikes, and evidence-based project assessments
- Multi-tool agent support (Claude Code, OpenCode, Cursor, Windsurf)
- LLM provider integration for the Dev Companion runner
- Windows support (WSL2 + Git Bash + skills-only)
- CI quality gates (MegaLinter v9, security scanning, pre-commit)
- Agentic harness framework with three-layer architecture
