# Linear Integration

> Linear issue, cycle, and project workflows via OAuth-based MCP.

---

## Best path

Use the Linear MCP server. No local API token is required.

## Setup

Register the streamable HTTP MCP endpoint from the template in `MCP`.

## What it enables

- Issues
- Cycles
- Projects

## Verify

1. Restart your AI tool.
2. Trigger a Linear action.
3. Approve the OAuth flow when prompted.
