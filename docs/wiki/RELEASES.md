# Releases & Updates

> Keep your workstation baseline and skills up to date.

---

## Check for updates

```bash
dots-update-check
```

## Apply updates

```bash
chezmoi update
dots-doctor
```

## Skills-only updates

Re-run the latest release installer for your platform:

```bash
curl -fsSL https://github.com/ulises-jeremias/dots-ai/releases/latest/download/install-skills.sh | sh
```

## Before opening a support issue

```bash
dots-doctor --issue
```
