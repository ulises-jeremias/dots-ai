# GitLab Integration

> GitLab authentication and merge request workflows.

---

## Best path

Use `glab` for GitLab operations.

## Setup

```bash
glab auth login
glab auth status
```

## What to use it for

- Merge requests
- Issues and repository browsing
- CLI-first GitLab automation

## Verify

```bash
glab auth status
dots-doctor
```
