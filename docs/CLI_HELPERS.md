## Maintainer contract

The workstation installs helper commands with the `dots-` prefix. Important entry points include:

- `dots-doctor`
- `dots-skills`
- `dots-loadenv`
- `dots-devcompanion`
- `dots-update-check`
- `dots-bootstrap`

Keep implementation details in `home/dot_local/bin/` and user workflows in the wiki.
