# Task (WHAT)

Use for technical or operational work that is not best represented as a user story.

## Default guardrails

1. Apply **`dots-ai-output-handshake`** before final output.
2. Use **`dots-ai-planning`** for estimation, dependencies, and breakdown.
3. Use **`dots-ai-bug`** for defects and **`dots-ai-incident`** for urgent production impact.
