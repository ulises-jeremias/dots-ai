# Epic (WHAT)

Use when a large body of work needs grouping into related stories, tasks, bugs, or subtasks.

## Default guardrails

1. Apply **`dots-ai-output-handshake`** before final output.
2. Use **`dots-ai-planning`** for estimation and breakdown.
3. Use **`clickup-cli`** for ClickUp creation only after user approval.
